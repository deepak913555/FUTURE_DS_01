Global E-Commerce Sales Performance Dashboard (2024)
Project Overview
This project focuses on analyzing e-commerce sales transaction data to identify key business insights such as total revenue, best-selling products, sales trends, and regional performance.

Objectives
Identify best-selling products and categories

Analyze monthly sales trends

Understand regional and payment method performance

Create KPIs to track overall business health

Present insights using interactive visuals

Key Insights
Electronics category contributes the highest revenue

Sales show clear seasonal trends

North America is the top revenue-generating region

Credit Card is the most preferred payment method

A small set of products drives a large share of total sales

Conclusion
The dashboard helps stakeholders quickly understand performance, trends, and growth opportunities.
